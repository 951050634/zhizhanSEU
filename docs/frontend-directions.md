# 前端视觉方向收敛

## 方向 A：展厅指挥墙

视觉关键词：

- 深石墨底色
- 琥珀与冰蓝双高光
- 大字号标题与展项编号
- 仪表化信息分区
- 像展厅总控台而不是普通网站

适用原因：

- 非常适合同时承载主展示、状态、事件和控制四块内容
- 能把“中控”而不是“宣传页”的产品气质做得很明确
- 后续接入真实 `WebSocket` 与设备状态后，扩展成本最低

## 方向 B：数字档案册

视觉关键词：

- 米白纸面
- 深蓝与朱红强调色
- 编目式排版
- 展品卡片、年份刻度与材料标签
- 更接近策展册与研究档案

适用原因：

- 很适合讲“展品内容”与“知识解释”
- 视觉上更温和，适合偏人文或校史场景
- 但它对状态区、事件区和控制区的承载力弱于中控方向

## 当前选型

本阶段 `web/` 第一版采用方向 A，也就是“展厅指挥墙”。

选定理由如下：

- 当前任务重点不是做单页宣传，而是做可联调的网页中控
- 页面必须同时容纳模拟消息、状态回显、操作控制和主展示内容
- 后续板子到手后，方向 A 更容易平滑过渡到真实多屏联动界面
